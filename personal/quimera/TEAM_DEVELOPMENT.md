# Quimera - Equipo de Desarrollo

## Composicion

### Python Backend Developer
**Agente:** `python-backend-developer`
**Responsabilidad:** FastAPI backend, servicios de generacion (LLM, images, TTS, PDF), integracion MercadoPago/Stripe.
**Stack:** Python 3.11+, FastAPI, SQLAlchemy, Pydantic, asyncio.

### Frontend React Developer
**Agente:** `frontend-react-developer`
**Responsabilidad:** PWA mobile-first, landing pages, flujo de compra, compartir en redes.
**Stack:** Next.js o Astro, TypeScript, TailwindCSS.

### Database Expert
**Agente:** `database-expert`
**Responsabilidad:** Schema PostgreSQL, modelos de usuario/perfiles/suscripcion, optimizacion de queries.
**Stack:** PostgreSQL 17, Redis.

### Test Engineer
**Agente:** `test-engineer`
**Responsabilidad:** Tests unitarios, integracion, E2E. Validacion de filtros de contenido.
**Stack:** pytest (backend), Jest/Playwright (frontend).

### Code Reviewer
**Agente:** `code-reviewer`
**Responsabilidad:** Review riguroso pre-PR, foco en seguridad (menores), calidad de codigo.
**Modo:** Harsh reviewer (directiva 8).

### DevOps Engineer
**Agente:** `devops-engineer`
**Responsabilidad:** CI/CD, deployment Railway/Fly.io, Docker setup (PG, Redis), monitoring.
**Stack:** GitHub Actions, Docker, Railway/Fly.io.
