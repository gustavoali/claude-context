## Descripcion

Capacitarse en Odoo (ERP open source). Evaluar nivel de profundidad: funcional, tecnico (desarrollo de modulos), o ambos.

## Puntos a definir

- **Enfoque:** Funcional (configuracion, procesos de negocio) vs Tecnico (desarrollo Python/OWL, modulos custom)
- **Version:** Odoo 17+ (Community vs Enterprise)
- **Material:** Odoo Academy, documentacion oficial, cursos externos
- **Certificacion:** Odoo tiene certificacion oficial (funcional y tecnica)
- **Practica:** instalar instancia local con Docker para experimentar

## Notas

- Odoo esta basado en Python + PostgreSQL, hay experiencia con ambas tecnologias
- El frontend de Odoo usa OWL (propio) y esta migrando desde jQuery
