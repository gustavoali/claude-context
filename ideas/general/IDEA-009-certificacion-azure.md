## Descripcion

Preparar y rendir certificacion de Microsoft Azure. Definir cual certificacion apuntar y armar plan de estudio.

## Puntos a definir

- **Nivel:** Fundamentals (AZ-900), Associate (AZ-204 Developer, AZ-104 Admin), o Expert
- **Path recomendado:** AZ-900 primero si no hay experiencia previa con Azure
- **Material de estudio:** Microsoft Learn, cursos, practice exams
- **Timeline:** definir fecha objetivo para rendir

## Notas

- Experiencia existente en .NET puede complementar bien con AZ-204 (Azure Developer Associate)
- Evaluar si el empleador actual cubre costos de certificacion
