## Descripcion

TinyClaw es un framework multi-agente, multi-equipo y multi-canal para asistentes IA 24/7 en Node.js. Permite ejecutar multiples equipos de agentes que colaboran entre si con workspaces aislados. Soporta Discord, WhatsApp y Telegram, procesamiento paralelo, dashboard TUI en vivo, y operacion 24/7 via tmux.

## Motivacion

- Referencia para orquestacion multi-agente (relevante para Claude Orchestrator)
- Patron de workspaces aislados por equipo
- Integracion con canales de mensajeria (WhatsApp, Telegram, Discord)
- Dashboard TUI como inspiracion para UIs de terminal
