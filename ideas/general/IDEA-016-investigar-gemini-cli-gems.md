## Descripcion

Investigar el ecosistema de Google Gemini: Gemini CLI como herramienta de desarrollo y Gems como mecanismo de personalizacion/especializacion.

## Puntos a investigar

- **Gemini CLI:** capacidades, comparativa con Claude Code, tool use, integracion con IDEs
- **Gems:** que son, como crearlos, casos de uso, limitaciones
- **API de Gemini:** modelos disponibles, pricing, context window, multimodalidad
- **Integracion:** compatibilidad con MCP, posibilidad de usar junto a Claude
- **Ventajas competitivas:** que ofrece Gemini que no tenga Claude y viceversa

## Notas

- Util para tener perspectiva del mercado y evaluar si hay herramientas complementarias
- Comparar experiencia de desarrollo con Claude Code vs Gemini CLI
