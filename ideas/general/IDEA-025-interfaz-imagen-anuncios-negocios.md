# IDEA-025: Interfaz de imagen barata para anuncios de pequenos negocios

**Fecha:** 2026-02-21
**Categoria:** general
**Estado:** Seed
**Prioridad:** Sin definir

---

## Descripcion

Herramienta o servicio accesible y economico que permita a pequenos negocios (kioscos, almacenes, peluquerias, talleres, etc.) crear material visual de calidad para sus anuncios y promociones, sin necesidad de conocimientos de diseno ni contratar a un disenador.

## Motivacion

Los pequenos comercios necesitan material visual (carteles, flyers, posts para redes sociales, banners) pero no tienen presupuesto para disenadores ni habilidades para usar herramientas como Photoshop o Illustrator. Terminan usando Word, Paint o fotos del celular con texto encima. Hay una brecha entre lo que necesitan y lo que pueden pagar/hacer.

## Posibles enfoques

- **App mobile:** el comerciante saca foto del producto/local, elige estilo, la app genera el anuncio
- **Web app simple:** templates predefinidos por rubro (gastronomia, indumentaria, servicios), arrastra foto y texto
- **IA generativa:** usar modelos de generacion de imagen para crear anuncios a partir de una descripcion de texto
- **Servicio por WhatsApp/Telegram:** manda foto + texto, recibe anuncio armado
- **Kiosko fisico:** terminal en papelerias/librerias donde imprimen el material

## Diferenciacion vs Canva/similares

- Mucho mas simple (3-4 pasos maximo, no un editor completo)
- Orientado a un publico no-tech (comerciante de barrio)
- Precio accesible o freemium con opciones muy baratas
- Templates especificos para negocios argentinos/latam
- Posibilidad de impresion directa (vinculacion con imprentas locales)

## Formato del producto

- App mobile (Android priority) o PWA
- Flujo: elegir rubro -> elegir template -> subir foto -> escribir texto -> descargar/compartir/imprimir
- Output: imagen lista para imprimir (A4/A3) o para redes sociales (IG, FB, WhatsApp status)

## Modelo de negocio posible

- Freemium: templates basicos gratis, premium por suscripcion mensual baja
- Pay per use: pago por anuncio generado con IA
- B2B: vender a imprentas/papelerias como servicio de valor agregado
- Publicidad local dentro de la plataforma

## Notas

- Investigar si hay algo similar orientado a Latam / Argentina
- El canal WhatsApp podria ser el MVP mas rapido (bot + IA)
- Considerar si esto podria ser un proyecto informatico o un emprendimiento con equipo
