**Fecha:** 2026-02-12
**Categoria:** general
**Estado:** Seed
**Prioridad:** Sin definir

## Descripcion

Investigar el ecosistema de OpenAI: Codex como agente de coding, modelos GPT, y evaluar si hay integracion o compatibilidad con MCP.

## Puntos a investigar

- **Codex:** capacidades como coding agent, comparativa con Claude Code y Gemini CLI
- **GPT (o1, o3, GPT-4.5, etc.):** modelos disponibles, fortalezas, pricing
- **OpenAI API:** function calling, assistants API, tool use, threads
- **MCP + OpenAI:** si OpenAI soporta MCP o hay bridges/adapters comunitarios
- **Custom GPTs:** comparar con Gems de Gemini y con el approach de agentes de Claude
- **Integracion practica:** posibilidad de usar MCP servers existentes (SBM, LTE) con clientes OpenAI

## Notas

- Completaria el panorama de los 3 grandes (Claude, Gemini, OpenAI) para tomar decisiones informadas
- Relacionada con IDEA-011 (Agentic AI), IDEA-012 (MCP) y IDEA-016 (Gemini)
