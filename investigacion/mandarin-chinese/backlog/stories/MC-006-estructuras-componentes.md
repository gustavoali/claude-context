# MC-006: Documentar las posiciones estructurales de los componentes
**Points:** 3 | **Priority:** Medium | **Epic:** EPIC-02

**As a** estudiante de mandarin
**I want** conocer las estructuras espaciales en que se organizan los componentes dentro de un caracter
**So that** pueda reconocer y descomponer visualmente cualquier caracter

## Acceptance Criteria

**AC1: Estructuras principales documentadas**
- Given que los componentes se organizan en patrones espaciales predecibles
- When documento las estructuras
- Then cubro al menos: izquierda-derecha, arriba-abajo, envolvente, envolvente parcial, superpuesto
- And cada estructura tiene diagrama visual + 5 ejemplos

**AC2: Frecuencia relativa de cada estructura**
- Given que no todas las estructuras son igualmente comunes
- When documento las estructuras
- Then incluyo la frecuencia aproximada de cada tipo
