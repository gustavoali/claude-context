# MC-015: Definir metodologia personal de estudio
**Points:** 1 | **Priority:** Low | **Epic:** EPIC-05

**As a** estudiante de mandarin
**I want** una metodologia personal de estudio documentada
**So that** pueda mantener progreso consistente y medible

## Acceptance Criteria

**AC1: Plan de estudio basico**
- Given que necesito estructura para avanzar
- When defino la metodologia
- Then incluyo: frecuencia de estudio sugerida, orden de temas, metricas de progreso
- And el plan es realista para alguien con tiempo limitado

**AC2: Integracion con el proyecto**
- Given que este proyecto de investigacion alimenta el aprendizaje
- When defino la metodologia
- Then explico como las notas de investigacion se conectan con la practica activa
