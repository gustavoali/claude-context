# LLM Landscape - Equipo de Desarrollo

## Nota

Este es un proyecto de investigacion de tipo documental (Markdown). No tiene desarrollo de software en la primera fase.

Si en fases futuras se decide agregar scripts Python (ej: scraping de precios de API, actualizacion automatica de benchmarks), los agentes relevantes seran:

| Tarea | Agente |
|-------|--------|
| Scripts Python de recopilacion de datos | `python-backend-developer` |
| Tests de los scripts | `test-engineer` |
| Code review de scripts | `code-reviewer` |

## Colaboracion Actual

Todo el contenido es generado por Claude en sesiones directas con el usuario, sin delegacion a agentes de desarrollo.
