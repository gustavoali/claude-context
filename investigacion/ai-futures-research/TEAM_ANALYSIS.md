# Equipo de Analisis - AI Futures Research

## Composicion
Este proyecto es primariamente de investigacion y analisis, no de desarrollo de software.

| Rol | Agente | Responsabilidad |
|-----|--------|-----------------|
| Coordinador | Claude (principal) | Dirigir investigacion, analisis critico |
| Investigador Web | general-purpose | Buscar fuentes, papers, articulos |
| Extractor de Texto | Python scripts | Extraer contenido de PDFs |

## Notas
- No requiere agentes de desarrollo (no hay codigo)
- El usuario es el decision-maker sobre que fuentes priorizar
- Claude produce analisis y sintesis basados en las fuentes
