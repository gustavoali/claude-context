# Project Management Tools - Seed Document
**Fecha:** 2026-03-21
**Origen:** Solicitud del usuario (sin documento fuente)

## Objetivo
Aprender herramientas de gestion de proyectos, comenzando por Trello. El proyecto es expandible para incluir otras herramientas (Jira, Asana, Monday, Linear, etc.) en el futuro.

## Alcance Inicial: Trello

### Que es Trello
- Herramienta de gestion de proyectos basada en tableros Kanban
- Modelo: Boards > Lists > Cards
- Freemium (plan gratuito suficiente para aprender)
- Propiedad de Atlassian

### Roadmap de Aprendizaje - Trello Basico

#### Fase 1: Fundamentos (30 min)
- Crear cuenta en trello.com
- Entender la interfaz: boards, lists, cards
- Crear un board de prueba
- Crear listas (To Do, In Progress, Done)
- Crear cards y moverlas entre listas
- Agregar descripcion, checklist, due dates a cards

#### Fase 2: Funcionalidades Intermedias (30 min)
- Labels (etiquetas de color para categorizar)
- Members (asignar personas a cards)
- Attachments (adjuntar archivos)
- Comments (comunicacion en cards)
- Activity log (historial de cambios)
- Filtros y busqueda

#### Fase 3: Power-Ups y Automatizacion (30 min)
- Butler (automatizacion nativa de Trello)
  - Rules (cuando X pasa, hacer Y)
  - Card buttons y board buttons
  - Calendar commands
  - Due date commands
- Power-Ups populares:
  - Calendar view
  - Custom Fields
  - Voting
- Integraciones (Slack, Google Drive, etc.)

#### Fase 4: Trello API y Automatizacion Avanzada (opcional, 1-2h)
- API REST de Trello
- Webhooks
- Integracion con scripts propios
- Posible MCP server para Trello

### Expansiones Futuras
- Jira (gestion de proyectos enterprise, Atlassian)
- Asana (gestion de tareas y proyectos)
- Monday.com (work OS)
- Linear (gestion de proyectos para equipos de software)
- Notion (como herramienta de PM, ya conocida)
- Comparativa entre herramientas
