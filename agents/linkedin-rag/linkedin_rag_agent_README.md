# LinkedIn Video RAG Agent (Fixed)

Versión con hotfix para resolver rutas de archivo descargadas por yt-dlp y notebook que valida el `video_path` automáticamente.

## Pasos rápidos
1) `pip install -r requirements.txt`
2) Exportá `cookies.txt` desde tu navegador (logueado en LinkedIn).
3) Abrí `notebooks/LinkedIn_RAG_Agent.ipynb`, cambiá `URL` y `COOKIES`, ejecutá.
